ARCHITECTURE INSPECTOR

Game

Central Game Controller

This class acts as the orchestration layer of the entire dungeon crawler. It coordinates rendering, inventory, maps, movement, crafting, and player state.

Connected Systems

  • Player
  • Inventory
  • MapManager
  • Renderer
  • CraftingSystem

Design Notes

The architecture intentionally radiates outward from Game to visually communicate that all gameplay systems are coordinated through one central engine.
