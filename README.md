# Dungeon Crawler OOP Game

## Description

A simple Java dungeon crawler game.

The player moves through levels, collects resources, crafts items, finds keys, breaks walls, and reaches the door.

## How to Run

GUI mode: `java Main`

CLI mode: `java Main -cli`

## Controls

W - move up  
A - move left  
S - move down  
D - move right  
E - use item  
C - craft  
I - inventory  
1 / 2 - equip item  
Q - quit CLI mode
